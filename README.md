# id-multi-label-hate-speech-and-abusive-language-detection
The Dataset for Multi Label Hate Speech and Abusive Language Detection in Indonesian Twitter

## How to cite us
This dataset and the other resource can be used for free, but if you want to publish paper/publication using this dataset, please cite this publication:

**Muhammad Okky Ibrohim and Indra Budi. 2019. Multi-label Hate Speech and Abusive Language Detection in Indonesian Twitter. In *ALW3: 3rd Workshop on Abusive Language Online*.** (Accepted and to be published).
